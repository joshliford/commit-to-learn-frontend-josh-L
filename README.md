## üìù Learning Flow

1. **JavaScript** ‚Üí Core language fundamentals: variables, functions, arrays, objects, classes.  
2. **HTML** ‚Üí Structure of webpages, semantic elements, forms, and accessibility.  
3. **CSS** ‚Üí Styling webpages: selectors, layouts (flexbox, grid), responsive design.  
4. **DOM (Document Object Model)** ‚Üí Connecting JavaScript to HTML: events, element selection, and manipulation.  
5. **JSON (JavaScript Object Notation)** ‚Üí A lightweight data format for sending and receiving structured data.  
6. **APIs (Application Programming Interfaces)** ‚Üí Using `fetch` to send HTTP requests (GET, POST) and work with external data.  
7. **React** ‚Üí A JavaScript library for building reusable UI components with JSX, props, and state.
